# PRD: 인생타이쿤 (LifeTycoon Kids)

> v1.0 | 2026-04-08 | 기준 커밋: 63+

---

## 1. 제품 개요

### 1.1 한 줄 요약
초등 고학년(10~13세) 대상, 10분 안에 10세→100세 인생을 경영하는 웹 기반 경제 육성 게임.

### 1.2 핵심 가치
- **다마고치 × 인생게임 × 캐시플로** 하이브리드
- 가상 주식/예금/직업으로 자산관리하며 "꿈(미션)" 달성
- 좌절 없는 어린이 친화적 톤 (게임오버 없음, 실패=시나리오 분기)
- 서버 없는 정적 사이트 (개인정보 수집 없음)

### 1.3 타겟 사용자
| 구분 | 상세 |
|------|------|
| 핵심 타겟 | 초등 4~6학년 (10~13세) |
| 확장 타겟 | 중학생, 경제교육에 관심 있는 부모 |
| 난이도 | 중1급 경제 개념 + 초등 친화 어휘 |
| 플랫폼 | 웹 (모바일 우선, PC 지원) |

## 2. 핵심 게임 루프

```
[타이틀] → [꿈 선택(1~3개)] → [게임 플레이(10분)]
                                    │
                          ┌─────────┴─────────┐
                          │  배속 시간 진행     │
                          │  (1년 ≈ 6.7초)     │
                          │                    │
                          │  ┌──────────────┐  │
                          │  │ 중요 이벤트   │  │
                          │  │ → 시간 멈춤   │  │
                          │  │ → 선택 대기   │  │
                          │  └──────────────┘  │
                          │                    │
                          │  투자/저축/케어     │
                          └─────────┬─────────┘
                                    │
                          [100세 → 엔딩]
                          (비문 + 등급 + 공유)
```

## 4. 콘텐츠 구성

### 4.1 시나리오 (250개)

| 카테고리 | 개수 | 설명 |
|---------|------|------|
| 일반 | ~175 | 일상, 학교, 직업, 건강, 관계 등 |
| 코믹 | ~40 | 떡볶이 토네이도, 외계인, 고양이 대통령 등 |
| 체인 | ~30 | 특성 조합 → 후속 이벤트 트리거 |
| 히든 | ~14 | 극단적 조건(부자/빈곤/특성 콤보) |
| 전설 | 5 | 3특성 이상 콤보, 극히 드문 발생 |

나이대별 커버리지 (해당 나이대에서 발생 가능한 시나리오 수):

| 나이대 | 커버 수 | 시작 기준 |
|--------|--------|----------|
| 10대 | 79 | 79 |
| 20대 | 141 | 83 |
| 30대 | 169 | 38 |
| 40대 | 176 | 16 |
| 50대 | 153 | 9 |
| 60대 | 116 | 9 |
| 70대 | 80 | 6 |
| 80대 | 58 | 5 |
| 90대 | 35 | 5 |

> "커버 수"는 ageRange가 해당 나이대를 포함하는 시나리오 수. 하나의 시나리오가 여러 나이대에 걸칠 수 있음.

### 4.2 체인 이벤트 주요 경로

```
코딩 배우기 → IT 스타트업 → 떡볶이 앱 → 100만 DL
인플루언서 → 악성 댓글 → 멘탈강철 → 강연 섭외
반려인 + 봉사왕 → 동물 천재 TV 출연
유주택 → 집값 폭등
발명가 → 시간여행자 방문
시간여행 경험 → 미래 복권 번호 (5천만 잭팟!)
절친 → 동업 → 대성공
레모네이드 → 카페 창업
코딩 + 인플루언서 + 발명 → 테크 아이콘 (5억!)
기부왕 + 봉사왕 + 의리맨 → 한국의 간디
```

### 4.3 가상 종목 (10개)

| 종목 | 섹터 | 배당 | 변동성 | 교육 포인트 |
|------|------|------|--------|-----------|
| 떡볶이 제국 🌶️ | 분식 | 2% | 35% | 친숙한 대상 투자, 체인 이벤트 핵심 |
| 무지개 전자 🌈 | 기술 | 1% | 25% | 성장주 투자 |
| 펭귄 택배 🐧 | 물류 | 3% | 18% | 안정+배당 균형 |

| 로켓김밥 🚀 | 분식 | 0% | 45% | 고수익=고위험 교훈 |
| 고양이 은행 🐱 | 금융 | 5% | 10% | 금융주+최고배당 |
| 메타우주여행 🌌 | 엔터 | 0% | 55% | 투기 vs 투자 경계 |
| 레전드 치킨 🍗 | 식품 | 2.5% | 22% | 중간 위험·배당 |
| 보라보라 화장품 💜 | 뷰티 | 1.5% | 28% | 트렌드 기반 투자 |
| 댕댕이 사료 🐕 | 반려 | 3% | 15% | 안정 배당 소형주 |

## 5. 교육적 가치

### 5.1 경제 개념 체험

| 개념 | 게임 내 구현 |
|------|------------|
| 복리 | 은행 예금 이자 누적 |
| 배당금 | 주식 보유 시 연간 수입 |
| 분산투자 | 여러 종목 보유 + 자동투자 |
| 인플레이션 | "물가 상승" 시나리오 |
| 비상금 | "비상금 만들기" 시나리오 |
| 세금 | "세금 폭탄" 시나리오 |
| 연금 | 65세+ 자동 수입 |
| ETF | "ETF 추천" 시나리오 |
| 기회비용 | 모든 선택지의 트레이드오프 |

### 5.2 인성 교육

- 정직함 (주운 돈 신고 → "정직함" 특성)
- 나눔 (기부 → "기부왕" 특성 → 히든 이벤트)
- 용기 (괴롭힘 대응 → "용감함" → 동네 영웅 체인)
- 건강 관리 (케어 버튼 + 자연 감소)
