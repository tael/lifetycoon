# 인생타이쿤 🎮

10세부터 100세까지 약 10분에 플레이하는 어린이 경제 육성 게임.

## 특징

### 핵심 메카닉
- **다마고치 × 인생게임 × 캐시플로** — 캐릭터 육성 + 배속 타임라인 + 경제 시뮬레이션
- **109개 시나리오** — 병맛/감성/교육/히든/체인 이벤트
- **10개 가상 종목 주식** — 배당금 시스템 포함 (떡볶이 제국, 로켓김밥, 메타우주여행 등)
- **12개 직업, 8개 꿈, 연금** — 매 플레이마다 다른 인생 경로
- **NPC 라이벌 4명** — 실시간 자산 랭킹 + 직접 상호작용 이벤트
- **다마고치 케어** — 스탯 자연감소 + 간식/건강/공부/노래 돌봄 버튼

### 게이미피케이션
- **영구 업적 시스템(10개)** + 하이스코어 기록
- **10년 주기 마일스톤** 인생 점검 팝업
- **꿈 달성 진행도 바** + 토스트 알림 + 컨페티
- **연 수입 분해** (월급 + 이자 + 배당 + 연금)
- **뉴스 티커** + 경제 교육 팁

### 엔딩
- **묘지 비문** — 인생 서사가 담긴 비석 + 타이핑 애니메이션
- **인생 한줄 요약** — 특성 기반 한 문장 표현
- **자산 추이 차트** — SVG 라인 그래프
- **NPC 비교 랭킹** + 인생 통계 대시보드
- **다음 인생 전략 팁** — 맞춤형 조언
- **URL 공유코드** — 친구에게 엔딩 공유

### UX
- **첫 플레이 튜토리얼** (4단계)
- **캐릭터 말풍선** (상태/나이/랭킹 기반 20+개 대사)
- **키보드 단축키** (1/2/3으로 선택)
- **모바일 진동** + 모달 애니메이션
- **이벤트 효과 미리보기** 아이콘
- **타이틀 격언** 로테이션

## 기술 스택

- React 19 + TypeScript + Vite + Zustand
- 정적 사이트 (서버 없음, ~113KB gzip)
- localStorage 저장 + URL 공유코드

## 실행

```bash
pnpm install
pnpm dev
```

## 빌드

```bash
pnpm build   # dist/ 에 정적 파일 생성
pnpm preview # 로컬 프리뷰
```

## 시나리오 추가

`src/game/data/scenarios.json`에 JSON 형식으로 추가:

- `id`, `triggers`, `ageRange`, `weight`, `pausesGame`
- `title`, `text`, `choices[{label, effects, importance}]`
- `tags`, `oneShot`
- 체인 이벤트: `triggers`에 `hasTrait` 조건 활용
